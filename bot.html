AI検索について

仕組み

AI検索は、Google の Gemini API を使って質問の「意図」を解析し、ローカルにキャッシュされたイベント・メンバー・実験データの中から最適な結果を返します。

個人情報や具体的なデータは AI に送信されません。送信されるのは質問文のみで、データ検索はすべてブラウザ内で行われます。

安全性

  • 質問文のみがサーバー（GAS プロキシ）経由で Gemini API に送信されます
  • メンバー名・個人情報は AI に送信されず、ブラウザ内で処理されます
  • API キーはサーバー側で管理され、クライアントには公開されません
  • 1日あたりの使用回数に上限があり、不正利用を防止しています

API について

Google Gemini API の無料枠を使用しています。1日あたりの上限に達すると、自動的にキーワード検索に切り替わります。上限は日本時間17時頃にリセットされます。

使い方のヒント

  • Enter で送信、Shift + Enter または Option(Alt) + Enter で改行
  • 回答のコピーは、メッセージ横の「コピー」ボタンで可能です
  • AI が使えない場合は自動でキーワード検索に切り替わります
